# Chess App Frontend

This is the frontend for the University Chess Rating System. It connects to the FastAPI backend to provide a seamless chess playing experience.

## Features

- **User Authentication**: Login and Register.
- **Game Interface**: Interactive chessboard with drag-and-drop.
- **Real-time Updates**: Uses WebSockets to sync game state.
- **Game History**: View past games and move logs.

## Tech Stack

- **React**: UI Library.
- **Vite**: Build tool.
- **Chess.js**: Chess logic validation.
- **React-Chessboard**: Visual board component.
- **Axios**: API requests.
- **TailwindCSS** (or custom CSS): Styling.

## getting Started

1. `npm install`
2. `npm run dev`
